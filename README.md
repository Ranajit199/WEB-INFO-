# WEB-INFO-
This tool can be used to extract information from any website

## Install Termux and run the command ---

$ apt-get update

$ apt-get upgrade

$ pkg install git

$ pkg install python

$ pkg install python2

$ git clone https://github.com/Ranajit199/WEB-INFO-

$ cd WEB-INFO

$ chmod +x *

$ python2 web-info.sh
